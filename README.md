# usage
You need to edit src/const.ts.

```
$ git clone https://github.com/w-okada/trtc-voice-visualization.git -b qiita-advent-2022
$ cd trtc-voice-visualization/
$ npm install
# edit src/const.ts
$ npm run build:worklet
$ npm run start
```
